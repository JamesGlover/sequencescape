#size(*args) ⇒ Object

By default rails treats sizes for grouped queries different to sizes for ungrouped queries. Unfortunately plates could end up performing either. Grouped return a hash, for which we want the length otherwise we get an integer We need to urgently revisit this, as this solution is horrible. Adding to the horrible: The :all passed in to the super is to address a rails bug with count and custom selects.
